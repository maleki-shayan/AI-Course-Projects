## Project Details

### 1. Search Agents in Pacman (CA1)
**Topic:** State Space Search & Heuristics  
This project involves designing intelligent agents for the classic **Pacman** game. The goal was to implement various search algorithms to help Pacman navigate the maze and eat all dots efficiently.

- **Algorithms Implemented:**
  - Breadth-First Search (BFS)
  - Depth-First Search (DFS)
  - Uniform Cost Search (UCS)
  - A* Search (A-Star) with custom heuristics
  - Weighted A* Search
- **Key Concepts:** Graph Traversal, Heuristic Design, Optimality vs. Complexity.

### 4. Deep Learning & Neural Networks (CA4)
**Topic:** Neural Networks, CNNs, & Sentiment Analysis  
A comprehensive introduction to Deep Learning using **PyTorch**. This project consists of multiple tasks involving image classification and text analysis.

- **Tasks:**
  - **PyTorch Warmup:** Basic tensor operations and gradient descent.
  - **Computer Vision:** Implementing a **Convolutional Neural Network (CNN)** for image classification.
  - **NLP:** Performing **Sentiment Analysis** on the SnappFood comments dataset.
- **Key Concepts:** Backpropagation, Activation Functions, Convolution, Loss Functions.

### 5. Unsupervised Learning (CA5)
**Topic:** Clustering & Dimensionality Reduction  
This project focuses on finding hidden patterns in unlabeled datasets using unsupervised learning techniques.

- **Tasks:**
  - Data Preprocessing and Visualization.
  - **Clustering:** Implementing algorithms like K-Means or DBSCAN.
  - **Dimensionality Reduction:** Using PCA (Principal Component Analysis) to visualize high-dimensional data.
- **Key Concepts:** Feature Extraction, Elbow Method, Silhouette Score.
